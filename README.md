# IDEIA
- Fazer um bomberman
- Cada fase terá um tempo limite para ele encontrar a saída
- Os níveis serão criados automaticamente

# Dependências
- Java Swing
- Java
- Java Develop Kit

# Paradigmas
- Classes Abstratas
- Polimorfismo (Tipos diferentes de bomba)
- Encapsulamento
- Herança (Power-up aumenta o tempo, Power-up de velocidade)
- Padrões de Projeto (Façade :- JOptionPane)
