# Organização dos pacotes
Foi dividio em dois diretório, o diretório bin para os arquivos binários e o diretóio src, para códigos fonte.\
O arquivo Main.java, contém a classe principal que rodará o jogo, outras classes serão criadas dentro de outros pacotes.\
Para compilar, executar o jogo, limpar os arquivos binários ou testar uma classe individualmente, usa-se a automação build.sh (para Linux) e build.bat (Para Windows)
